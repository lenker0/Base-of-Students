Распакуйте содержимое архива в любое место ( Рабочий стол подойдет идеально )
Чтобы открыть программу вам нужно найти файл  Laba2Rabochaya(A ne ta fignya).exe его путь указан ниже:
Laba2Rabochaya(A ne ta fignya)\Laba2Rabochaya(A ne ta fignya)\bin\Release

Вы зашли в программу, у вас вылезло окно логин т.к. вас нет в базе, то вводите
login - admin
password - admin

Браво, вы в базе данных, следующие что нужно сделать, это открыть саму базу студентов 
File->Open->Students.xml ( в изначальной папке находится данный файл ) 

Теперь вы можете Добавить студента - "+"
Удалить - "Delete"
Отсортировать по какому-либо признаку "Sort" или же дабл клик по столбцу, которому хотите отсортировать
Справа сверху есть поиск по ФИО ( Регистр важен ) 
Нажав на кнопку Compare и выбрав любую фото из папки bmp, вы можете найти студента с данной "цифровой подписью"
Также добавить новую цифровую подпись "Add image"

File -> Save - Сохранить
